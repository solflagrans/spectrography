# Спектрографический калькулятор

**Современный интерфейс для анализа спектра и поиска вероятного состава образца**

От сырых измерений до понятной картины: график, пики, совпадения с элементами и готовый отчет в одном рабочем пространстве.

</div>

---

## Суть

**Спектрографический калькулятор** превращает набор длин волн и интенсивностей в наглядную историю о составе образца. Приложение помогает увидеть значимые пики, сопоставить их с базой спектральных линий и быстро понять, какие элементы могут присутствовать в данных.

Это не просто форма для расчетов, а компактная исследовательская панель: данные, параметры, график, таблица пиков и вывод по элементам собраны в одном месте, чтобы анализ ощущался цельным и управляемым.

## Главное на экране

| Блок | Что пользователь видит |
| --- | --- |
| **Спектр** | Чистый график с линией спектра, порогом и найденными пиками |
| **Итог анализа** | Вероятные элементы, оценка совпадений и короткий вывод по образцу |
| **Пики** | Таблица линий с фильтрацией, сортировкой и отклонением от базы |
| **База линий** | Редактор элементов и опорных спектральных линий |
| **Отчет** | Скачиваемый JSON с результатами анализа |

## Пользовательский сценарий

### Загрузить спектр

Можно начать с демонстрационного пресета или добавить собственные данные. Приложение поддерживает JSON-файл и ручной ввод двух массивов: длин волн и интенсивностей.

### Настроить чувствительность

Порог, допуск совпадения, сглаживание и расстояние между пиками помогают адаптировать анализ под разные данные: от аккуратных учебных примеров до более шумных экспериментальных измерений.

### Увидеть результат

После анализа калькулятор показывает найденные пики на графике, собирает совпадения по элементам и выделяет наиболее уверенные результаты. Вместо длинного ручного сравнения пользователь получает ясную картину в несколько кликов.

### Проверить детали

Каждый пик можно рассмотреть отдельно: длина волны, интенсивность, предполагаемый элемент, ближайшая спектральная линия и отклонение. Это удобно, когда нужно объяснить результат или проверить спорные совпадения.

## Возможности

- Импорт пользовательских измерений;
- Автоматический поиск значимых пиков;
- Сопоставление пиков с базой химических элементов;
- Интерактивная настройка параметров анализа;
- Фильтрация и сортировка найденных линий;
- Редактирование базы спектральных линий;
- Экспорт результатов в отчет.

## Данные для анализа

Приложению нужны две связанные последовательности: длины волн в нанометрах и соответствующие им интенсивности.

```text
Длины волн, нм:
486.13
589.59
656.28

Интенсивности:
72.4
91.8
76.0
```

Те же данные можно передать в JSON:

```json
{
  "wavelengths": [486.13, 589.59, 656.28],
  "intensities": [72.4, 91.8, 76.0]
}
```
